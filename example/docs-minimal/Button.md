# Button

A reusable button component with various styles and sizes @example <Button variant="primary" size="lg" onClick={() => console.log('clicked')}> Click me </Button> /

- **variant** (`'primary' | 'secondary' | 'danger'`)

- **onClick** (`() => void`)
